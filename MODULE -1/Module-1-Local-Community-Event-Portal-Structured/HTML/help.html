Help Center

Learn how to use the Local Community Event Portal.

How to Use This Portal

View Events

Use the Events section to see event details, date, location, category, fee and available seats.

Search Events

Use the category filter and search box to find events easily.

Register

Click Register near an event, fill the form and submit your details.

Other Events

Use the Other Events section to load extra events from the events.json file.

Registered Events

After submitting the form, your registered event details will be displayed.

Feedback

Use the Feedback section to share comments about community events.

Project File Notes

  • HTML files are stored inside the HTML folder.
  • CSS files are stored inside the CSS folder.
  • JavaScript files are stored inside the JAVASCRIPT folder.
  • events.json is kept separately in the main project folder.
  • Open the project using Live Server for fetch and events.json to work properly.
